# 项目作品

这里展示我的个人项目，包括项目介绍、技术栈和开发笔记。

## 项目列表

### 个人博客系统

基于 Docusaurus 的学习博客，用于记录学习笔记和代码片段。

**技术栈：**
- React
- TypeScript
- Docusaurus
- GSAP

**主要功能：**
- 学习笔记管理
- 代码片段展示
- 项目作品展示
- 学习日志记录
- 亮暗主题切换
- 响应式设计

### 待办事项应用

全栈待办事项管理工具，支持用户注册、任务管理和数据统计。

**技术栈：**
- Vue 3
- Node.js
- Express
- MongoDB

**主要功能：**
- 用户注册和登录
- 任务创建、编辑、删除
- 任务分类和标签
- 数据统计和图表
- 响应式设计
