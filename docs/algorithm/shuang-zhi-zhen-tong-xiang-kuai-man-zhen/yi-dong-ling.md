## 一、题目描述

**题目：** 给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

**注意：** 必须在不复制数组的情况下原地对数组进行操作。

**示例 1:**

输入: nums = \[0,1,0,3,12\]  
输出: \[1,3,12,0,0\]

**示例 2:**

输入: nums = \[0\]  
输出: \[0\]

## 二、题解

### 思路：

1.  设置两个指针，一个快指针用于遍历数组判断元素是否为零，一个慢指针用于记录下一个插入非0元素的位置。
    
2.  当快指针遍历到的元素不为0的值时，将慢指针记录的位置赋值为值，并将慢指针向右移动一个单位。
    
3.  当快指针遍历完数组后，将慢指针以及之后的位置元素都赋值为0 。
    

### 代码：

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int slow = 0;
        for(int i = 0;i < nums.length;i++){
            if(nums[i] != 0){
                nums[slow] = nums[i];
                slow++;
            }
        }
        for(int i = slow;i < nums.length;i++){
            nums[i] = 0;
        }
    }
}
```
