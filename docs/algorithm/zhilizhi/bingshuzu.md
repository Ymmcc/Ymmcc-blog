## 一、题目描述

**题目：** 给你两个按 **非递减顺序** 排列的整数数组 nums1 和 nums2，另有两个整数 m 和 n ，分别表示 nums1 和 nums2 中的元素数目。

**要求：** 请你 **合并** nums2 到 nums1 中，使合并后的数组同样按 ** 非递减顺序** 排列。

**注意：** 最终，合并后数组不应由函数返回，而是存储在数组 nums1 中。为了应对这种情况，nums1 的初始长度为 m + n，其中前 m 个元素表示应合并的元素，后 n 个元素为 0 ，应忽略。nums2 的长度为 n 。

**示例 ：**

输入：nums1 = \[1,2,3,0,0,0\], m = 3, nums2 = \[2,5,6\], n = 3  
输出：\[1,2,2,3,5,6\]  
解释：需要合并 \[1,2,3\] 和 \[2,5,6\] 。  
合并结果是 \[**1**,**2**,2,**3**,5,6\] ，其中斜体加粗标注的为 nums1 中的元素。

## 二、题解

### 思路：

1.  由于要求**原地合并（即不能用新数组）** ，所以应该从后往前合并，可以设置 nums1 有效数组最右边指针为 p1，nums2 数组最右边指针为 p2，nums1 整个数组最有边指针（写入指针）为 writePos 。
    
2.  将 nums1\[p1\] 与 nums2\[p2\] 比较，谁大谁写入 nums1\[writePos\] 中，然后移动相应的指针继续遍历。
    
3.  直到将 nums2 数组元素移动完（即 p2 < 0），就完成了数组的合并。
    

### 代码：

```java
class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int p1 = m - 1,p2 = n - 1,writePos = m + n - 1;
        while(p2 >= 0){
            if(p1 >= 0 && nums1[p1] > nums2[p2]){
                nums1[writePos--] = nums1[p1--];
            }else{
                nums1[writePos--] = nums2[p2--];
            }
        }
    }
}
```
